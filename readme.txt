Show most popular posts of the current month.

== Description ==

Simple Wordpress Plugin that gives you the ability to add a widget where the most popular posts for the current month is displayed. You also have the option to name the widget and define number of posts to show.
Read more about it at: http://byme.se/wp-plugins/top-posts-this-month.html

== Installation ==

1. Upload the folder to YOUR-URL.com/wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the widgets panel and drag and drop the widget to your sidebar and configure it if you like
